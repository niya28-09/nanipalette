Nani Palette (AI Image Genrator)
This project generates images using AI based on your prompts.

Technologies Used
-Python, 
-html, 
-css, and Hugging face API.

Description
Nani Palette allows users to enter a text prompt and generate images based on the input.  The application processes the prompt and produces AI-generated images through a web interface.
